## 2. RECOPILACIÓN Y USO DE DATOS

### 2.1 Datos Personales Recopilados

La Aplicación recopila los siguientes datos personales proporcionados voluntariamente por el usuario:

- **Nombre:** Para personalizar la experiencia del usuario
- **Edad:** Para cálculos de salud personalizados
- **Género:** Para recomendaciones específicas
- **Altura:** Para calcular el Índice de Masa Corporal (IMC)
- **Peso inicial:** Para establecer punto de referencia
- **Peso objetivo:** Para calcular progreso hacia la meta
- **Registros de peso:** Histórico de mediciones realizadas por el usuario
- **Fecha de cada registro:** Para seguimiento temporal

### 2.2 Almacenamiento Local

**IMPORTANTE:** Todos los datos personales se almacenan **ÚNICAMENTE EN EL DISPOSITIVO DEL USUARIO** mediante tecnología localStorage del navegador. 

Esto significa que:
- ✅ Los datos **NO se envían** a ningún servidor externo
- ✅ Los datos **NO se comparten** con el Responsable
- ✅ Los datos **NO se almacenan** en la nube
- ✅ El Responsable **NO tiene acceso** a los datos del usuario
- ✅ Los datos permanecen **100% privados** en el teléfono del usuario

### 2.3 Finalidad del Tratamiento

Los datos recopilados se utilizan exclusivamente para:

1. Calcular el Índice de Masa Corporal (IMC)
2. Realizar seguimiento del progreso de peso
3. Generar gráficas de evolución
4. Proporcionar recomendaciones de alimentación y ejercicio personalizadas
5. Enviar notificaciones de recordatorio (si el usuario las activa)
6. Mejorar la experiencia del usuario

## 7. DERECHOS DEL USUARIO

Conforme a la **Ley 1581 de 2012** de Colombia (Ley de Protección de Datos Personales) y el **Decreto 1377 de 2013**, el usuario tiene los siguientes derechos:

### 7.1 Derecho de Acceso

El usuario puede acceder a sus datos en cualquier momento desde la Aplicación, en la sección "Perfil" e "Historial".

### 7.2 Derecho de Rectificación

El usuario puede modificar sus datos directamente en la Aplicación:
- Nombre, edad, género, altura → Sección "Perfil"
- Registros de peso → Editar o eliminar en "Historial"

### 7.3 Derecho de Supresión

El usuario puede eliminar sus datos de las siguientes formas:

**Opción 1: Borrar datos individuales**
- Eliminar registros de peso uno por uno en "Historial"
- Restablecer perfil en "Configuración"

**Opción 2: Borrar todos los datos**
- Configuración → "Restablecer Aplicación"
- Esto elimina TODOS los datos permanentemente

**Opción 3: Desinstalar la aplicación**
- Al desinstalar, todos los datos locales se eliminan automáticamente

**IMPORTANTE:** Al eliminar datos, **NO se pueden recuperar** ya que no existen respaldos en servidores.

### 7.4 Derecho de Oposición

El usuario puede oponerse al procesamiento de sus datos:
- Desactivando Google Analytics (contactando al Responsable)
- Desactivando anuncios personalizados (Configuración de Android)
- Dejando de usar la Aplicación

### 7.5 Derecho de Portabilidad

El usuario puede exportar sus datos en formato JSON o CSV:
- Herramientas → "Exportar Datos"
- Genera archivo con todos los registros
- El usuario puede importar en otra aplicación
